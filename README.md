# Convertio

Browser-based image converter PWA. Convert, resize, and optimize images entirely in your browser — no server uploads, complete privacy.

**[Live Demo](https://hawkeyepierce89.github.io/convertio/)**

## Features

- **Format Conversion** — Convert between JPEG, PNG, and WebP
- **Input Support** — Accept JPEG, PNG, WebP, GIF, and BMP files
- **Resize** — Change dimensions with optional aspect ratio preservation
- **Quality Control** — Adjust output quality (10-100%)
- **Privacy First** — All processing happens client-side using Canvas API
- **PWA** — Install as an app, works offline
- **Multi-language** — English, Russian, German, Spanish, Chinese

## Tech Stack

- TypeScript
- Vite
- Canvas API
- vite-plugin-pwa (Workbox)
